# GITHUB SEARCH

### Descrição
aplicação para consulta de perfil no github(para fins de estudo)

### Contexto de negócio
```
React
Typescript
TailwindCSS
```

### requisitos
```
node v16.13.2
```


## Executar localmente
```
# clone o repositório, entre na pasta do projeto
# npm install
# npm run dev
```


#### endereços para acesso:
```
Endereço da aplicação: http://localhost:5173/
```
